Was ist das hier?

Dies ist eine Webanwendung, die es ermöglicht, Musik zu verwalten und zu organisieren. Sie ist für den privaten Gebrauch gedacht.

Die Anwendung ist in Python geschrieben und nutzt das Flask-Framework. Die Daten werden in einer SQLite-Datenbank gespeichert.

Die Anwendung ist Open Source und kann auf GitHub gefunden werden.

Was kann die Anwendung?

Die Anwendung kann Musik verwalten. Es können Songs, Alben und Interpreten angelegt werden. Außerdem können Playlists erstellt werden, die entweder manuell oder automatisch mit Songs gefüllt werden können.

Was kann die Anwendung nicht?

Die Anwendung kann keine Musik abspielen. Sie kann auch keine Musik herunterladen. Sie kann auch keine Musik erkennen. Sie kann keine Musik empfehlen. Sie kann auch keine Musik verkaufen.
